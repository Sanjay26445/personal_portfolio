## Quick Start

### Prerequisites
- Node.js (v14 or higher)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:5173`

### Build

```bash
npm run build
```

### Deploy

```bash
npm run deploy
```
